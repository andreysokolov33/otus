## Настройка PaGP протокола между L3 коммутаторами

Для повышения надежности сети между L3 коммутаторами сделан агрегированный канал, через который, в случае неработоспособности одного оператора связи, трафик будет перенаправлен на другого оператора.

В 1 агрегированный канал будут входить порты gi1/0/1-3 на CORE-1SW и CORE-2SW:

```
interface range gi1/0/1-3
channel-group 1 mode desirable

interface Port-channel1
switchport mode trunk
switchport trunk native vlan 573
switchport trunk allowed vlan 55-57,60,77,80,88-93,100,111,125,130,200
```

Выбран режим desirable, чтобы TRUNK образовывался сразу, если есть такая возможность. В пункте [настройки маршрутизации](./routing_settings.md) через агрегированный канал будет проложен запасной маршрут по умолчанию, если один из операторов связи отвалится.
